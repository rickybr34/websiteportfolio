# Portfolio Website

Personal portfolio website built with React and Vite.

## Project Structure

```
├── client/          # Frontend React application
│   ├── src/         # Source code
│   ├── public/      # Static assets
│   └── vite.config.js
├── server/          # Backend server (future)
└── package.json     # Root package.json
```

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Tech Stack

- React 19
- JavaScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Deployed to Vercel
